# Basic Use

## Import

<Snippet
  file="getting-started/import-icon.jsx"
  expand
>
  Import the `Icon` component from `@abw/badger-icon`.
</Snippet>

## Insert

<Snippet
  file="getting-started/render-icon.jsx"
  expand
>
  Insert the `Icon` component wherever you want an icon. The `name` should
  be the name of an icon in the default icon set, an icon that you've added
  to the default library, or is in a custom library you've constructed
  yourself.
</Snippet>

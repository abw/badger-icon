# Modifiers

## Move

The `left=N`, `right=N`, `up=N` and `down=N` suffixes can be used to move the
icon within the view box. Each number corresponds to 1/16th of the icon size.
For example, `right=8` will move the icon by 50% to the right.

In these examples we&apos;re adding a border around the icon (using the
`border` CSS class) so you can more easily see the effect.

# Icon

## `type`

The `type` property can be used to set style parameters for the icon.  The
default value for `type` is `fill` which sets the fill color to be
`currentColor`.  Try toggling between <SetLight/> and <SetDark/> themes to
see the fill color change.

<Example
  Component={Fill}
  code={FillSource}
  undent={4}
  expand
/>

The `line` style will set the stroke color to `currentColor` and the `fill`
to `none`.

<Example
  Component={Line}
  code={LineSource}
  undent={4}
  expand
/>

The `type` can contain multiple <Link to="/modifiers" text="modifiers"/>
separated by hyphens.  For example, we can set the stroke weight to be
`thin`...

<Example
  Component={Thin}
  code={ThinSource}
  undent={4}
  expand
/>

...or `thicker` with `square` corners and end caps.

<Example
  Component={Thick}
  code={ThickSource}
  undent={4}
  expand
/>

As a short-cut, the `type` can be provided as a prefix on the `path`,
separated by a colon.

<Example
  Component={Path}
  code={PathSource}
  caption="path"
  undent={4}
  expand
/>

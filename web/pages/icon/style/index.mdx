# Icon

## `style`

The `style` property can be used to apply custom styling to the icon.

## Style String

The `style` can be an SVG property string. Properties can be written in
`camelCase` or `kebab-case`.

<Example
  Component={StyleStr}
  code={StyleStrSrc}
  undent={4}

/>

## Style Object

The `style` can also be an object.  Properties can be written in `camelCase`
or `kebab-case`.

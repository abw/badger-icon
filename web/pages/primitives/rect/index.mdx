# Primitives

## `rect`

The `rect` element can be used to draw rectangles with square or rounded
corners.

The explicit form is to define the icon as an object with the `element`
property set to `rect`.  Additional properties should be included for `x`,
`y`, `width`, `height` and optionally, `rx` and `ry`. Any other properties
can be included as attributes for the generated SVG `rect` element.

<MDNElementLink element="rect"/>

<Example
  Component={Component}
  code={Source}
  expand
/>

The short form of this is an object with a `rect` property which is a string
containing the parameters for `x`, `y`, `width`, `height` and optionally,
`rx` and `ry`.

The even shorter form is to define the icon as a string with the `rect`
prefix, a colon, and then the parameters for `x`, `y`, `width`, `height`,
and optionally, `rx` and `ry`.

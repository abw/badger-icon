# Primitives

## `ellipse`

The `ellipse` element can be used to draw ellipses.

The explicit form is to define the icon as an object with the `element`
property set to `ellipse`.  Additional properties should be included for `cx`,
`cy`, `rx` and `ry`. Any other properties can be included as attributes for
the generated SVG `ellipse` element.

<MDNElementLink element="ellipse"/>

<Example
  Component={Component}
  code={Source}
  expand
/>

The short form of this is an object with an `ellipse` property which is a
string containing the parameters for `cx`, `cy`, `rx` and `ry`.

<Example
  Component={Short}
  code={ShortSrc}
  expand
  undent={2}
/>

The even shorter form is to define the icon as a string with the `ellipse`
prefix, a colon, and then the parameters for `cx`, `cy`, `rx` and `ry`.

<Example
  Component={String}
  code={StringSrc}
  expand
  undent={2}
/>

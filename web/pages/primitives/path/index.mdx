# Primitives

## `path`

The `path` element is the generic element to define a shape.

The explicit form is to define the icon as an object with the `element`
property set to `path`.  The `d` property should define the data points.
Any other properties can be included as attributes for the generated SVG
`path` element.

<MDNElementLink element="path"/>

<Example
  Component={Component}
  code={Source}
  expand
/>

The short form of this is an object with a `path` property which is a string
containing the `d` path data.

<Example
  Component={Short}
  code={ShortSrc}
  expand
  undent={2}
/>

The even shorter form is to define the icon as a string with the `path`
prefix, a colon, and then the path data.

<Example
  Component={String}
  code={StringSrc}
  expand
  undent={2}
/>

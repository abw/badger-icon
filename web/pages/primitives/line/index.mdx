# Primitives

## `line`

The `line` element can be used to draw straight lines.

The explicit form is to define the icon as an object with the `element`
property set to `line`.  Additional properties should be included for `x1`,
`y1`, `x2` and `y2`. Any other properties can be included as attributes for
the generated SVG `line` element.

<MDNElementLink element="line"/>

<Example
  Component={Component}
  code={Source}
  expand
/>

The short form of this is an object with a `line` property which is a string
containing the parameters for `x1`, `y1`, `x2` and `y2`.

<Example
  Component={Short}
  code={ShortSrc}
  expand
  undent={2}
/>

The even shorter form is to define the icon a string with the `line` prefix,
a colon, and then the parameters for `x1`, `y1`, `x2` and `y2`.

<Example
  Component={String}
  code={StringSrc}
  expand
  undent={2}
/>
